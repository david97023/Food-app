# Food-app
「美食app平台」  

功能簡介
1. 主要介面：  
在app的介面中，會有一個登入介面確認使用者身分，進入後可以選擇「美食清單、健康管理、算帳高手、地址搜索、記事本、留言板」六種功能，美食清單中可以使用進階搜尋讓使用者做設定，像是：金額限制、熱門指數……等，點選後就可以知道該店家的詳細資訊；運用地址搜索可以立即用關鍵字找到正確的地址；健康管理可以計算 BMI 值並提供一些小建議；算帳高手能夠讓你輕鬆地計算數學或折扣；記事本則可以協助記錄下重要事項。
2. 背景音樂：
將app背景放入音樂，讓使用者可以一邊使用app找尋喜歡的店家，一邊聆聽著好聽的音樂放鬆心情，音樂也可以隨著流行做出更動，同時完全不需要擔心會找不到好吃的餐廳。
3. 留言板：
大多數關於店家的優缺點和細節，使用者在網路上的官網介紹上是看不出來的，但是實際體驗過的人絕對是最清楚，因此特地設置了留言板來讓使用者們回饋他們自己的意見和想法，提供使用者一個回饋心得及評價的良好平台，讓其他用戶有值得參考的真實評價。
